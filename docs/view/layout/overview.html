Basic Layout

You can define a basic layout using:

  • .layout define the main container.
  • .layout-sidebar define the layout sidebar. Fixed left, hidden on smalls devices.
  • .layout-header define the layout header. Fixed top.
  • .layout-body define the layout body container. Auto scrollable.
  • .layout-footer define the layout footer. Fixed bottom.

Style

Feel free to use the classes bg-*-* for background color and text-*-* for text color, defined on Color Palette.

  • You can resize the window to show/hide sidebar.
  • Elements with .layout-toggler will be hidden when sidebar were visible.
  • Use .toolbar to styles header.
  • The container is auto-scrollable.
  • Sidebar shows if tou swipe from left to right.

Sidebar

You can create a sidebar NAV and sidebar ACCOUNT BOX.

  • .layout-nav define styles for link navigation.

Create multiples navs using .nav-hidden and use [data-toggle="nav"] with [data-target="navID"] to toggle the visibility.
