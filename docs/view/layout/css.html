Positioning

Type

  • .static sets position: static.
  • .relative sets position: relative.
  • .absolute sets position: absolute.
  • .fixed sets position: fixed.
  • .sticky sets position: sticky.

Axis

  • .top sets top: 0.
  • .right sets right: 0.
  • .bottom sets bottom: 0.
  • .left sets left: 0.
  • .top-inverse sets top: 100%.
  • .right-inverse sets right: 100%.
  • .bottom-inverse sets bottom: 100%.
  • .left-inverse sets left: 100%.

Util

Floating

  • .pull-left sets float: left.
  • .pull-right sets float: right.

Set-*

  • .set-border sets border: 1px solid #ddd.
  • .set-radius sets border-radius: 3px.
  • .set-circle sets border-radius: 50%.
  • .set-overlay sets box-shadow: inset 0 0 100px 10px rgba(0, 0, 0, .5).

No-*

  • .no-margin sets margin: 0.
  • .no-padding sets padding: 0.
  • .no-float sets float: none.
  • .no-border sets border: none.
  • .no-shadow sets box-shadow: none.
  • .no-transition sets transition: none.
  • .no-bg sets background: none.
  • .no-style sets background: transparent; border: 0; color: inherit; font-size: inherit; font-family: inherit; list-style: none; margin: 0; outline: none; padding: 0.
