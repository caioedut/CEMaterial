Events
Event name Occurs when
cem.datepicker.beforeShow Datepicker dialog is called, but not opened yet
cem.datepicker.show Datepicker dialog is opened
cem.datepicker.beforeHide Datepicker dialog close action is triggered, but not closed yet
cem.datepicker.hide Datepicker dialog is closed

Basic

Datepicker dialog can be shown by using [data-toggle="datepicker"].

Colors

You can change the color using the Color Palette with [data-color="red-6"]

Min/Max Date

You can set a min/max date using [data-min="X"] and [data-max="X"], X can be:

  • Explicit date 2017-01-01;
  • Integer value, gets the current date and adds X days: 7;
  • Negative value, get the current date and subtract X days: -7.

Custom Handler

You can use custom elements (not a input) to handle the datepicker, like a button or an icon, using the [data-toggle="datepicker"] and [data-target="#inputDate"]
