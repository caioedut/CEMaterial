Basic

Buttons are defined by the class .btn. Using .btn-flat you get smaller paddings.

Colors

Feel free to use the classes bg-*-* for background color and text-*-* for text color, defined on Color Palette.

Icons and circle

You can use the class .btn-icon if the button is only an icon and the class .btn-cicle to round the corners perfectly.

Elevation

It is to display a shadow behind the button.

Group

To gruoup multiple buttons.

Sizes

You can use the classes .btn-xs, .btn-sm, .btn-lg and .btn-xl.

Set the button full width using .btn-block
