Basic

You can show a toast using the attributes [data-toggle="toast"] with [data-toast="Message"] defined.

Advanced

Actions

You can add action buttons to a toast, using javascript declaration. Actions must have these options:

  • label (string) - Button text.
  • color (string) - Button text color, you can use the colors defined on Color Palette.
  • onClick (function) - The action triggered when button was clicked.
